Short summary:

This agent is tuned to perform targeted code-logic reviews in the opened
folder. It prioritizes correctness over style and will suggest minimal,
actionable fixes. To begin, ask the agent to "Run a code-logic audit" or
"Find async/race-condition issues".

Suggested follow-ups:

- Create a companion `code-logic-checklist.md` for recurring checks.
- Add a CI job that runs selected static checks and unit tests for automation.
